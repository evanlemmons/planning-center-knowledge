# Planning Center Glossary

A running list of terminology, jargon, and acronyms used at Planning Center and in church technology.

## Planning Center Products

| Term | Definition |
|------|------------|
| **People** | Free membership database — the foundational product which backs the people profiles available in all other products. All member data lives here. Includes forms and workflows. |
| **People Mobile** | Mobile app specifically for People admins. Does not send push notifications. Possibly being deprecated soon if PCA gains traction. |
| **Home** | Free ecosystem-level dashboard hub, task management. Always included and accessible for every user. |
| **Planning Center App / PCA** | The upcoming unified mobile app which sends push notifications for all admin web notifications and leverages an in-app browser to navigate Planning Center pages. Built in React Native. |
| **Tasks** | Ecosystem-wide collaborative task management feature. Part of Home, accessible from any product via the taskbar, available natively in PCA |
| **Services** | Worship planning and rehearsal tools for musicians + volunteer scheduling for ministry teams |
| **Services Mobile** | Mobile app specifically for Services users. Push notifications for scheduling requests; interactive media player for rehearsal on the go. |
| **Music Stand** | Mobile app add-on for Services: sheet music display, annotations, foot pedal control |
| **Groups** | Community group management: events, attendance, shared resources, and Chat via Church Center |
| **Registrations** | Event signups, tickets, and payments (powered by Stripe) |
| **Giving** | Online/in-person donation processing, fund management, pledges, reporting |
| **Check-Ins** | Attendance tracking, child safety (printable name tags, security labels, emergency alerts) |
| **Check-Ins Mobile** | Mobile app for Check-Ins admins. Some event management, but primarily intended to be used as a remote check-in station. |
| **Headcounts** | Mobile app for Check-Ins admins. Allows for manually inputting attendance numbers for specific events to help track overall attendance. |
| **Calendar** | Event scheduling, facility booking, resource management, and approval systems for all aforementioned things |
| **Calendar Mobile** | A simple web wrapper around Calendar - NOT a true mobile app. Does not send push notifications. Is possible to access other products via the product switcher. Almost exactly like using Planning Center from a mobile browser. On the docket to be deprecated shortly after PCA launches. |
| **Church Center** | Congregant-facing mobile app + website powered by other PC products. Sometimes referred to as "CC" or "CCW" for "Church Center Web" to differentiate from CCA |
| **Church Center App / CCA** | The mobile app version of Church Center which sends congregant-facing push notifications |
| **Publishing** | Management system for Church Center: custom pages, sermon video/audio library, podcast syndication, service streaming |
| **Chat** | Team/group messaging accessible from any product via the sidebar and also in both Services Mobile and Church Center / CCA. Shown on Home marketing page but NOT actually part of Home, it's an ecosystem-level feature just like notifications and tasks |

## Feature/System Terms

| Term | Definition |
|------|------------|
| **PCO / PC** | Common abbreviation for Planning Center Online (the company) |
| **Ecosystem-level feature** | Feature available to all users for free regardless of subscription (e.g., Home, Chat, Notifications, Tasks) |
| **Church Center** | The member/congregant-facing layer of Planning Center; sometimes used to refer to both the app (CCA) and website (CC/CCW) |
| **CCLI** | Christian Copyright Licensing International — license for using copyrighted worship songs. Services integrates with CCLI for reporting. |
| **Auto-schedule** | Services feature that automatically fills volunteer positions based on preferences and availability |
| **PCU** | Planning Center University — video training courses for every product. Regularly updated. Recommended for all new users. planningcenter.com/university |
| **New for You / NFY** | Video series on the PC YouTube channel and social media summarizing recently released features and changes |
| **PC YouTube** | youtube.com/user/planningcenter — hosts PCU courses, "New for You" updates, and other video content |
| **Integration** | 3rd party products who've built on top of Planning Center's API. planningcenter.com/integrations |
| **Add-on (Verified)** | Integration partner that renders React UI inside PC products (e.g., Clearstream, MinistrySafe). Requires formal partnership with PC's API team. |
| **JSON:API** | The API spec PC uses (jsonapi.org). REST with includes, filtering, pagination. |
| **PicoLabs** | PC's innovation lab for experimental tools (e.g., NFC Links app). Limited support, may not all survive. |
| **Circle** | Platform hosting PC's official user community (community.planningcenter.com). Used for beta feedback spaces. |
| **PC FB Group** | facebook.com/groups/planningcentercommunity — private, community-run, NOT PC-owned. Most active user community. No direct API access. |

## Church Technology Terms

| Term | Definition |
|------|------------|
| **ChMS** | Church Management Software/System |
| **Congregation** | The membership/attendees of a church |
| **Campus** | A physical location of a church (multi-site churches have multiple) |
| **Service** | A worship gathering/event |
| **Volunteer** | Church members who serve in various roles |
| **Check-in** | Process of recording attendance, especially for children's ministry, may include printing a sticker/badge |
| **Small group** | Smaller gatherings for community, study, or fellowship. Also known as "Community Groups", "Life Groups", etc. |
| **Tithe** | Traditional 10% giving; sometimes used loosely for any donation to a church |
| **Offerings** | Non-tithe donations to a church |
