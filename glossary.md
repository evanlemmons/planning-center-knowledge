# Planning Center Glossary

A running list of terminology, jargon, and acronyms used at Planning Center and in church technology.

## Planning Center Products

| Term | Definition |
|------|------------|
| **People** | Free membership database — the foundational product. All member data lives here. |
| **Home** | Free ecosystem-level dashboard hub, task management, and toolbar. Always included. |
| **PCA** | Planning Center App — the upcoming unified mobile app (internal project) |
| **Tasks** | Ecosystem-wide task management feature (part of Home) |
| **Services** | Worship planning and volunteer scheduling for ministry teams |
| **Music Stand** | Mobile add-on for Services: sheet music display, annotations, foot pedal control |
| **Groups** | Community group management, attendance, and chat via Church Center |
| **Registrations** | Event signups, tickets, and payments (powered by Stripe) |
| **Giving** | Online/in-person donation processing, fund management, pledges, reporting |
| **Check-Ins** | Attendance tracking, child safety (name tags, security labels, emergency alerts) |
| **Calendar** | Event scheduling, facility booking, and resource management |
| **Church Center** | Member-facing mobile app + website powered by other PC products |
| **Publishing** | CMS for Church Center: custom pages, video/audio library, podcast syndication |
| **Chat** | Team/group messaging. Separate product with own team. Shown on Home marketing page but NOT part of Home. |

## Internal Terms

| Term | Definition |
|------|------------|
| **Ecosystem team** | Cross-functional team caring for platform-wide features and product cohesion |
| **Ecosystem-level feature** | Feature available to all users for free regardless of subscription (e.g., Home, Chat, Notifications) |
| **Product Pulse** | (Notion database) - TBD, need more context |
| **Church Center** | The member/congregant-facing layer of Planning Center (app + website) |
| **PCO** | Common abbreviation for Planning Center Online (the company) |
| **CCLI** | Christian Copyright Licensing International — license for using copyrighted worship songs. Services integrates with CCLI for reporting. |
| **Auto-schedule** | Services feature that automatically fills volunteer positions based on preferences and availability |
| **PCU** | Planning Center University — video training courses for every product. Recommended for all new users. planningcenter.com/university |
| **New for You** | Video series on the PC YouTube channel summarizing upcoming features and recent changes |
| **PC YouTube** | youtube.com/user/planningcenter — hosts PCU courses, "New for You" updates, and other video content |
| **Add-on (Verified)** | Integration partner that renders React UI inside PC products (e.g., Clearstream, MinistrySafe). Requires formal partnership with PC's API team. |
| **JSON:API** | The API spec PC uses (jsonapi.org). REST with includes, filtering, pagination. |
| **PicoLabs** | PC's innovation lab for experimental tools (e.g., NFC Links app). Limited support, may not all survive. |
| **Circle** | Platform hosting PC's official user community (community.planningcenter.com). Used for beta feedback spaces. |
| **PC FB Group** | facebook.com/groups/planningcentercommunity — community-run, NOT PC-owned. Most active user community. No direct API access. |

## Church Technology Terms

| Term | Definition |
|------|------------|
| **ChMS** | Church Management Software/System |
| **Congregation** | The membership/attendees of a church |
| **Campus** | A physical location of a church (multi-site churches have multiple) |
| **Service** | A worship gathering/event |
| **Volunteer** | Church members who serve in various roles |
| **Check-in** | Process of recording attendance, especially for children's ministry |
| **Small group** | Smaller gatherings for community, study, or fellowship |
| **Tithe** | Traditional 10% giving; sometimes used loosely for any donation to a church |
